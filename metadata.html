What is PDF Metadata?

Every PDF contains hidden properties (metadata) that describe the file. This includes information such as the Title, Author, Subject, Keywords, Creator tool, and creation date. This data is used by search engines, digital libraries, and file explorers to categorize and index the document.

Change Properties Privately

ToolsPDF allows you to instantly read and change these properties online. Because the modifications are processed via JavaScript inside your local device's memory, your document remains entirely private and is never stored on our servers.
